### Task 1.2: Configure Tailwind CSS and Shadcn UI
**Description**: Set up Tailwind CSS with custom configuration and install Shadcn UI components.

**Implementation Steps**:
- Configure `tailwind.config.ts` with custom color palette (primary blue, success green, error red, warning amber)
- Install Shadcn UI CLI: `npx shadcn-ui@latest init`
- Install required Shadcn components: Button, Card, Input, Textarea, Dialog, Dropdown Menu, Tabs, Badge, Avatar, Toast, Alert, Select, Calendar
- Create `components/ui/` folder for Shadcn components
- Set up global styles in `app/globals.css`

**Dependencies**: Task 1.1

**Files to Create/Modify**:
- `tailwind.config.ts`
- `components.json` (Shadcn config)
- `app/globals.css`
- `components/ui/*`
